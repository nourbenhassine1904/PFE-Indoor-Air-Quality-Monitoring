# PFE — Indoor Air Quality Monitoring

Platform for monitoring and improving indoor air quality using data analysis and ML.
Includes:
- Angular frontend (`Projet_Pfe/front`)
- Spring Boot backend (`Projet_Pfe/back`)
- Flask module (`Flask/projet_flask`)
- Data preparation / experiments (`TEST.py`)

## Tech Stack
Angular • Spring Boot • Flask (Python) • Machine Learning • Data Analysis

## Run locally (high level)
- Front: `cd Projet_Pfe/front` → `npm install` → `ng serve`
- Back: `cd Projet_Pfe/back` → run Spring Boot (Maven)
- Flask: `cd Flask/projet_flask` → create venv → install deps → `python app.py`

> Note: large files (datasets, Power BI dashboards, videos) are intentionally excluded from this repository.
